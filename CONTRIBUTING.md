# Contributing

Thanks for your interest! This project is intentionally small and
focused, so most contributions aren't needed. If you have a suggestion,
bug report, or improvement in mind, please **open an issue first**
before writing any code. Do not create unsolicited pull requests.

## Changing the injected prompt

The most opinionated part of this project is the prompt text that
the plugin injects into every subagent's first turn. If you have
an idea for a better wording:

- **Within the original design intent** — open an issue to discuss it.
- **Outside the original design intent** — please fork the project and
  adjust your own copy. Do not open issues or pull requests proposing
  a change that deviates from the intended scope.
